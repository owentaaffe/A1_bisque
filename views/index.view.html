The main part of the assignment is in the reservation page.

This uses 2 classes Customer.class.php and table.class.php

Within the form HERE to collect data about a reservation from a customer

The customer class collects info about the user i.e. Name/Email/Phone No.

The table class collects info about the reservation i.e. Date/Time/How many people and a message from the user.

When submitted a message relaying the info appears in the same page.

These can be joined together if using a database by adding a customer Id to the table.class.php

Each input is a required field, apart from the message box. The date field does not allow user to pick a past date and the time is a dropdown menu.

The menu pages are using their own classes

This is so extra methods can be added in the future i.e select choice from any menu for ordering/delivery

These classes are using Multidimensional arrays called from content/menu folder

This is so any changes to the menu can be made from one place

I have seperated the HTML and PHP files for each page

In the PHP file I include the HTML file,

So as to not have too many PHP opening/closing tags

Apart from some exceptions in the reservation.html file (date input), and the menu PHP files have echoed some html for their type and times.
